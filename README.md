# Rolling

## Intro

이 프로젝트는 `pnpm create vite` 명령어로 만들었습니다.

## Getting Started

- Install [pnpm](https://pnpm.io)

```sh
npm i -g pnpm
```

or

```sh
brew install pnpm
```

- Install dependencies

```sh
pnpm i
```

- Run development environment

```sh
pnpm start
```

- Go to `http://localhost:3000`
